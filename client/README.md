# LMS FRONTEND

### Setup Instructions

***********************************************************************
1. Clone the Proect from here

    git clone https://github.com/vandan501/LMS_MERNStack/tree/main

***********************************************************************    
2. Move into the Directory

    cd LMS_MERNStack 

***********************************************************************    
3. Install Dependencies

    npm install
***********************************************************************    
4. Run the Server

    npm run start
